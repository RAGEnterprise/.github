## Visão Geral

O RAGOS Linux é um sistema operacional leve, rápido e personalizável, com foco em automação declarativa e ambientes de alta performance.

Aqui você encontrará:

- Ferramentas oficiais do sistema  
- Infraestrutura de construção da ISO  
- Scripts de provisionamento  
- Repositório de pacotes adicionais  
- Projetos experimentais mantidos pela comunidade  

---

## Projetos em Destaque

- **ragos-builder** → Sistema oficial para gerar a ISO customizada  
- **ragos-pkgs** → Pacotes, aplicações e integrações oficiais  
- **ragos-installer** → Instalador automatizado  
- **ragos-docs** → Documentação oficial do sistema
